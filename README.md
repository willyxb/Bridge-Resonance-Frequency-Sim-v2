Bridge Resonance Frequency Simulator

An interactive physics simulation that demonstrates resonance, natural frequency, damping, and oscillatory motion in bridge structures. This project was developed as part of an AP Physics 1 project to help users visualize how periodic forces can cause dramatic increases in vibration amplitude when they match a structure's natural frequency.

Overview:

Bridge failures caused by resonance are among the most famous examples of applied physics in engineering. This simulator allows users to explore the relationship between external forcing frequency, natural frequency, and damping through an interactive model.

The project was designed to serve as an educational tool for students learning about simple harmonic motion, resonance, and real-world engineering applications.

Features: Interactive bridge vibration simulation

Adjustable forcing frequency

Adjustable damping effects

Real-time visualization of oscillatory motion

Demonstrates resonance and amplitude amplification

Educational explanations of key physics concepts
